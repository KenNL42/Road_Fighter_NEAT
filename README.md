# Road_Fighter_NEAT

The project is inspired by classic NES game Road Fighter. This project uses NEAT python to let AI evades obstacles

# Modules List

<li>NEAT python</li>
<li>Pygame</li>
<li>Matplotlib</li>
<li>pickle</li>
